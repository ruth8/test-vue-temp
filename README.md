# Vue 应用

基于 Vue 3 + Vite 的简单示例项目。

**环境要求：Node.js 18 或更高版本**（当前若为 Node 10 会报错，需先升级）。

### 如何升级 Node.js

- **使用 nvm（推荐）**：`nvm install 20` 然后 `nvm use 20`
- 或从 [nodejs.org](https://nodejs.org/) 下载安装 LTS 版本

安装完成后执行 `node -v` 确认版本 ≥ 18，再运行下面的命令。

## 开发

```bash
npm install
npm run dev
```

在浏览器中打开终端显示的本地地址（通常是 http://localhost:5173）。

## 构建

```bash
npm run build
```

## 预览构建结果

```bash
npm run preview
```
